# Multi-Agent Platform

一个可进化的多 Agent 协作平台设计稿，核心能力包括：
- 动态 Agent 管理与智能任务路由
- Skills 商店与 Skill 快速生成
- Agent-Skill 授权控制
- 经验包（EvoMAP）沉淀、检索、评分与复用
- 多 Agent 群聊协作与结果收敛
- 小龙虾助手（OpenClaw 风格）统一管理与调度

## 文档
- 设计方案：`design.md`
- 架构总览图：`architecture.puml`

## 如何查看架构图
如果本机已安装 PlantUML，可执行：

```bash
plantuml architecture.puml
```

生成的图片可用于方案评审或实现对齐。
